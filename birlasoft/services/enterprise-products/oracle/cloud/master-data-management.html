iLink
Solution Overview:

PiCoL: A Secure, Light Weight and Rich User Experience for Supplier Collaboration
Solution Overview:
Birlasoft PiCoL tool is a One stop shop Supplier Collaboration for PLM Cloud.
Challenges
Supplier Collaboration for Product Information
Supplier collaboration for product information and digital assets poses some key challenges, like no single source of truth, limited access to suppliers, and a simplified UI
No Single Source of Truth
Enterprises want to maintain a single source of truth for Product Attributes/Digital Assets/Bill of Material / Structure/Compliance Certificates and Packaging information
Lack of security controls
Business want to selectively expose the information to their suppliers. They would not want to expose the system of record to suppliers. Thus a granular security enforcement and a secondary lightweight interface is required.
Complex User Interfaces
Suppliers at the same time expect a simple user interface for accessing this information. User experience and intuitive interface are paramount to them
Solution Highlights
  • Allows suppliers to access product information and digital assets through a rich and intuitive user interface
  • Filters relevant information to honor granular security setup in Oracle PLM cloud
Simplify user interaction by:
  • Capturing attribute extension on a single page
  • Display associated digital assets at product and BOM level
  • Allows download of all digital assets
  • Quick deployment on Oracle PaaS platform
  • Uses REST and SOAP API and is ready to use
